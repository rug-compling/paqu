Soort invoer

Is de tekst in je corpus getokeniseerd?

Voorbeeld zonder tokenisatie: 

    <s><t>Dit is een zin.<t></s>
Voorbeeld met tokenisatie: 
    <s>
        <w><t>Dit</t></w>
        <w><t>is</t></w>
        <w><t>een</t></w>
        <w><t>zin</t></w>
        <w><t>.</t></w>
    </s>
Een FoLiA-bestand kan zowel getokeniseerde als niet-getokeniseerde zinnen bevatten, ook tegelijk. Maar PaQu kan maar één soort invoer tegelijk aan. Als je aangeeft dat de tekst getokeniseerd is wordt alle niet-getokeniseerde tekst genegeerd, en andersom.
