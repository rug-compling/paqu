FoLiA — Help

Upload metadata

Als je corpus verwijst naar externe metadata moet je dat apart uploaden.

Hier is een voorbeeld van een FoLiA-bestand dat verwijst naar externe metadata: 

<?xml version="1.0" encoding="UTF-8"?>                                                                          
<FoLiA xmlns="http://ilk.uvt.nl/folia" xmlns:xlink="http://www.w3.org/1999/xlink">
  <metadata src="voorbeeld.xml" type="cmdi">
    ...
  </metadata>
  <text>
    ...
  </text>
</FoLiA>

Externe metadata moet XML zijn, bijvoorbeeld cmdi of imdi, en gecodeerd zijn in UTF-8.

Je kunt een enkel metadatabestand uploaden, of meerdere in één tar- of zip-bestand. Als je opnieuw metadata uploadt wordt oude metadata verwijderd. Je kunt dus geen metadata toevoegen.

Wanneer je FoLiA-bestanden hebt geüpload in een tar- of zip-bestand, en daarin zijn paden gebruikt, dan moet je voor de metadatabestanden dezelfde paden gebruiken. Dus als je een FoLiA-bestand path/to/folia1.xml hebt met daarin een verwijzing naar metadata1.xml, dan moet je een metadatabestand path/to/metadata1.xml hebben.
